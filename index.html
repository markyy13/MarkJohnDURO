Serial Barcode Scanner


Serial Barcode Scanner is a web-based internship project that I created using PHP and the Vue.js framework. This innovative tool enables users to scan serial barcodes and part numbers of products, effortlessly storing this data in a database. Moreover, it boasts the functionality to track and timestamp product outputs when the same serial number and part number are scanned again, streamlining inventory management and enhancing efficiency.

Virtual Line


Virtual Line is my second web-based internship project, where I leveraged PHP and the Vue.js framework to create an innovative solution. This system employs a barcode scanner to read serial barcodes, which are linked to video records stored in a database. Upon scanning a barcode, the system automatically plays the corresponding product video, serving as a valuable tool for product presentations straight from the factory floor.

SSCR-DC Parking System


The SSCR-DC Parking System is our innovative Capstone project, meticulously crafted using PHP and the Laravel framework. While it currently serves as a prototype due to the absence of parking slot demarcations at our school, it has received school approval and holds the promise of future implementation once the parking facilities are renovated. This system exemplifies our dedication to advancing technology for practical solutions.
